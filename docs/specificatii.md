# Specificații Detaliate

Aplicația noastră este o platformă web care permite monitorizarea prezenței la evenimente. 
Organizatorii pot crea grupuri de evenimente, pot genera un cod de acces (text sau QR), 
pot vedea participanții și pot exporta rapoarte. 
Participanții introduc codul și își confirmă prezența.

---

##  Ce roluri există?
- **Organizator Eveniment**
- **Participant**

---

##  Ce funcții va avea aplicația?
- Crearea unui grup de evenimente  
- Adăugarea unui eveniment  
- Generarea unui cod (QR + text)  
- Introducerea codului de către participant  
- Afișarea participanților în timp real  
- Export CSV/XLSX  

---

##  Tehnologii folosite
- **Frontend:** React  
- **Backend:** Node.js + Express  
- **Bază de date:** MySQL  
- **ORM:** Prisma
